# Домашнее задание к работе 8
## Условие задачи
Вычислите вещественное значение 1 + 1/2 + 1/3 + ... + 1/N для целого N, заданного пользователем

## 1. Алгоритм и блок схема

### Алгоритм
1. Начало.
2. Ввод целого числа N.
3. Проверка: если N<=0, то вывести сообщение об ошибке и завершить программу.
4. Цикл от `i = 1` до `i = N`:
   - Вычислить `1 / i`.
   - Прибавить это значение к `sum`.
5. После завершения цикла вывести значение `sum`.
6. Конец.

## 2. Реализация программы
```
#include <stdio.h>
#include <locale.h>
int main() {
	setlocale(LC_CTYPE, "RUS");
	int N;
	double sum = 0.0;
	printf("Введите число N:");
	scanf("%d", &N);
	if (N <= 0) {
		printf("Ошибка: N должно быть положительным числом");
		return 1;
	}
	for (int i = 1; i <= N; ++i) {
		sum += 1.0 / i;
	}
	printf("=%.6f", sum);
	return 0;


}
```
